# FFMPEG-Youtube-dl-BatchMenu
Ugh, what's that command again?!  It's painful to look up ffmpeg and youtube-dl commands, so this tool does all the common stuff for you!

#Menu
1 - Download Video

2 - Download Playlist

3 - Split Video into Images

4 - Webm to MP4

5 - Split MKV

6 - MKV to MP4

7 - Concat mkv files

8 - Images to Video
